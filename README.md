## Argumento del modelo
Presentamos una estrategia que desarrolla un modelo de inteligencia articial destinado a predecir categorías a partir de datos de aceleración en tres ejes de dos acelerómetros ubicados en la espalda baja y el muslo derecho de adultos mayores por un timepo
aproximado de 40 minutos. Utilizamos un conjunto de datos compuesto por 15 muestras obtenidas del conjunto [Activity Recognition in Senior Citizens](https://www.kaggle.com/datasets/anshtanwar/adult-subjects-70-95-years-activity-recognition). 

Los archivos de este conjunto de datos se encuentran en la carpeta [Archivo](https://github.com/Concentracion-IA/Reto_1_Modelo_TC3006C/tree/main/Archivo).
